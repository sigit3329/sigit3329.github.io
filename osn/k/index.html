Kunci Sukses OSN-K

✓
  • ➤

    Simulasi Cermat

    Seringkali solusi terbaik didapat dengan mencoret-coret di kertas, mengikuti instruksi soal untuk N=1, N=2, N=3 hingga tuntas.

  • ➤

    Pola Matematis (Pattern Recognition)

    Jika batasannya N = 10^18, pasti ada polanya atau menggunakan rumus matematika O(1). Cari pola berulang.

  • ➤

    Reasoning (Penalaran)

    Gunakan Invariant dan Pigeonhole. Pikirkan skenario terburuk (worst case).

  • ➤

    Tracking State

    Pahami betul di posisi mana program sedang berjalan (variabel apa yang nilainya sedang berapa). Penting untuk debugging mental.

  • ➤

    Optimasi Sederhana

    Kuasai Prefix Sum, Sorting, dan Binary Search untuk mengubah kode 10 detik menjadi 0.01 detik.
